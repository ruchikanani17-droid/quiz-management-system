# Classic Indian TV Shows Web Page & Quiz

## What is this?
This is a front-end web project I put together using HTML, CSS, and JavaScript. It serves as a nostalgic tribute to classic Indian television, featuring an informational page about legendary shows[cite: 5] and an interactive quiz to test your memory[cite: 6]. I built this to practice designing user interfaces and writing JavaScript logic for an interactive web app.

## What it does
* **Showcases Classics:** Features a styled landing page with descriptions and images of iconic shows like *Malgudi Days*, *Hum Log*, *Shaktimaan*, and *Mahabharat*[cite: 5].
* **Interactive Quiz:** Includes a step-by-step, 7-question multiple-choice quiz about various classic Indian TV programs[cite: 6].
* **Input Validation:** The quiz uses JavaScript to alert the user and prevent them from skipping ahead if they haven't selected an answer[cite: 7].
* **Auto-Grading:** Automatically checks user selections against the correct answers and calculates the final score out of 7 at the end of the quiz[cite: 7].

## How I built it
I used standard web technologies to build this interface:
* **HTML:** Structured the main landing page and created the hidden/visible question containers for the quiz flow[cite: 5, 6].
* **CSS:** Styled the pages with a custom color palette, including a warm background, dark teal content containers, and interactive hover effects to make the UI feel modern[cite: 4, 8].
* **JavaScript & jQuery:** Wrote the script to navigate between questions, validate radio button inputs, and tally up the final score by comparing selected values against the correct answers[cite: 6, 7].
